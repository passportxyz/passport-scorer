Wielding the Ban Hammer

ALL field conditions must be met for a ban to apply.

Ban Types:

  1. Ban a Specific Credential
    Fill in: Hash only
    Example: Ban credential hash "0x123..." regardless of which address holds it
  2. Ban All Activity from an Address
    Fill in: Address only
    Example: Ban address "0xabc..." from using any stamps
  3. Ban Specific Provider for an Address
    Fill in: Both Address and Provider
    Example: Ban address "0xabc..." from using only the CoinbaseDualVerification stamp

Note: The form will reject an invalid configuration

Note: To ban multiple independent fields (e.g., to ban all credentials matching either an address OR a hash), create separate ban records

Revoke Matching Credentials

After creating a Ban, on the Ban list view page, you can use the Revoke Matching Credentials for selected ban(s) action to revoke all existing credentials that that match the selected ban(s)
